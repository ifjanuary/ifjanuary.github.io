创建hexo博客

最近在研究总结记录一些自己遇到的问题，之后就看到很多大佬们都会去写一些技术博客，作为小小白一枚，也跟了个风。

其实博客有很多，还是看自己如何选择了，自己搭建的这个是基于hexo框架搭的一个静态博客。选择的主题是next，原因很简单，就是这个主题很火，如果遇到什么问题只要一百度，基本就能解决，很方便。

主要分享下自己搭建博客的一个过程吧，其实网上也是有很详细的教程，自己参考的博文是这个：

https://blog.bestzuo.cn/posts/blog-establish.html

博主大大写的很详细，从搭建到next主题样式的修改都非常nice，要是有朋友将要搭建hexo博客或者是正在搭建遇到问题都可以参考参考这篇文章。

由于自己基础很薄弱，在做侧边菜单栏的时候也遇到了问题，然后看了下面这篇文章

https://blog.csdn.net/lijing742180/article/details/85622524

也解决了。

将本地推向远端的时候出现错误：hexo ERROR Deployer not found: github

解决办法的地址：https://www.jianshu.com/p/5c6c6cf325cd

自己目前也只是把框架和基本的界面理清楚了，对于其他的一些功能还没有完善，之后有时间会继续写的。
